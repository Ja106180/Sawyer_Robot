# 第1课：基础概念 - 为什么需要卡尔曼滤波？

## 🎓 学习目标
理解状态估计的基本概念和卡尔曼滤波的核心思想

## 📖 核心概念

### 1. 什么是状态估计？

**状态**：我们想知道的某个系统的真实情况
- 例如：小车的位置、速度、IMU的姿态角

**问题**：我们无法直接测量到"真实"状态
- 传感器有噪声
- 测量不完整
- 系统有不确定性

**解决方案**：用**估计**来逼近真实状态

### 2. 为什么需要滤波？

想象你在开车，GPS告诉你位置，但：
- GPS有误差（可能偏差几米）
- GPS更新频率低（每秒1次）
- 你还需要知道速度

**滤波的作用**：
- 融合多个信息源（GPS + 速度计）
- 减少噪声影响
- 预测未来状态

### 3. 卡尔曼滤波的核心思想

卡尔曼滤波做两件事：

#### 🔮 **预测（Prediction）**
- 根据**模型**预测下一个状态
- 例如：知道当前速度和位置，预测1秒后的位置
- **不确定性会增加**（因为我们不完美）

#### 📊 **更新（Update）**
- 用**测量值**来校正预测
- 例如：GPS告诉我们新位置
- **不确定性会减少**（因为有了新信息）

#### 🔄 **循环**
```
预测 → 更新 → 预测 → 更新 → ...
```

### 4. 一个生活例子

**场景**：你在房间里，想知道温度

- **模型预测**：如果现在是20°C，1分钟后可能还是20°C（变化慢）
- **测量**：温度计显示21°C（但有误差）
- **卡尔曼滤波**：结合两者，给出最佳估计（比如20.5°C）

### 5. 关键术语（先了解，后面会详细讲）

- **状态（State）**：我们想估计的量（位置、速度等）
- **预测（Prediction）**：根据模型预测
- **更新（Update）**：用测量值校正
- **协方差（Covariance）**：不确定性的度量
- **卡尔曼增益（Kalman Gain）**：决定相信预测还是测量

## 💡 理解要点

1. **卡尔曼滤波不是魔法**，它只是用数学方法融合信息
2. **预测有不确定性**，测量也有不确定性
3. **卡尔曼滤波找到最佳平衡点**，给出最优估计
